## 🚀 3단계 설정 (총 5분)

### 1단계 — 수동 테스트
터미널에서 작업 폴더로 이동 후 1회 실행:

```bash
cd "/Users/kimsun/프로그래밍/동학개미 지표2/동학개미지표2"
python3 telegram_daily.py
```

텔레그램 봇에서 메시지가 도착하면 성공입니다. `requests` 같은 외부 패키지 설치 불필요 (표준 라이브러리만 사용).

### 2단계 — launchd 등록 (매일 08:00 자동 실행)

```bash
# 1) plist를 LaunchAgents 폴더에 복사
cp "/Users/kimsun/프로그래밍/동학개미 지표2/동학개미지표2/com.donghak.telegram.plist" ~/Library/LaunchAgents/

# 2) 등록 (한 번만)
launchctl load ~/Library/LaunchAgents/com.donghak.telegram.plist

# 3) 즉시 1회 강제 실행해서 등록 확인
launchctl start com.donghak.telegram
```

이후 Mac이 켜진 상태라면 매일 아침 8시에 자동 실행됩니다.

### 3단계 — 티스토리 글 URL 추가 (선택)

`.env` 파일을 열어 `DASHBOARD_URL=` 뒤에 발행한 티스토리 글 주소를 넣으면, 메시지에 "대시보드에서 전체 보기 →" 링크가 함께 발송됩니다.

```
DASHBOARD_URL=https://본인티스토리.tistory.com/123
```

## 🔧 데이터 더 추가하려면

`telegram_daily.py` 안 `fetch_*` 함수를 추가하면 됩니다. 추천 확장 후보:

### FRED (미국 거시지표) — 무료 키 발급, 1분 소요
가입: https://fred.stlouisfed.org/docs/api/api_key.html

```python
FRED_KEY = os.environ["FRED_API_KEY"]
def fetch_vix():
    d = http_json(f"https://api.stlouisfed.org/fred/series/observations?series_id=VIXCLS&api_key={FRED_KEY}&file_type=json&sort_order=desc&limit=1")
    return float(d["observations"][0]["value"])
# 추가 시리즈: DGS10(10년물), DGS2(2년물), DFF(연준 기준금리), UNRATE(실업률), CPIAUCSL(CPI)
```

### Yahoo Finance — 키 불필요 (비공식)
```python
def fetch_sp500():
    d = http_json("https://query1.finance.yahoo.com/v8/finance/chart/^GSPC?range=2d&interval=1d")
    closes = d["chart"]["result"][0]["indicators"]["quote"][0]["close"]
    last, prev = closes[-1], closes[-2]
    return last, (last/prev - 1) * 100
```

### 한국 KOSPI / KOSDAQ
```python
def fetch_kospi():
    d = http_json("https://query1.finance.yahoo.com/v8/finance/chart/^KS11?range=2d&interval=1d")
    closes = d["chart"]["result"][0]["indicators"]["quote"][0]["close"]
    return closes[-1], (closes[-1]/closes[-2] - 1) * 100
```

## 🛠 운영·문제해결

### 자동 실행 중지
```bash
launchctl unload ~/Library/LaunchAgents/com.donghak.telegram.plist
```

### 로그 보기
```bash
tail -f "/Users/kimsun/프로그래밍/동학개미 지표2/동학개미지표2/telegram_daily.log"
```

### Mac이 꺼져 있던 시간대를 놓친다면
`com.donghak.telegram.plist`에서 `<key>RunAtLoad</key><false/>` 를 `<true/>` 로 바꾸면 부팅 직후에도 1회 실행됩니다 (단, 8시 이후 부팅 시 그 시점에 실행).

### 시간 변경
plist의 `<key>Hour</key><integer>8</integer>` 숫자만 바꾸고 `launchctl unload` → `load` 다시.
